# nuxt-ssr-firebase-startkit


## Requirement
- node v10.20.0

## Usage
1. Intro  
```bash
$ git clone https://github.com/y-ono366/nuxt-ssr-firebase-startkit.git
$ cd nuxt-ssr-firebase-startkit/functions
$ npm install
```  

2. Set firebase projects-name in `.firebaserc`

.firebaserc  
```bash
{
  "projects": {
    "default": "<projects-name>"
  }
}

```
